Path file adalah alamat lengkap atau lokasi file di situs web. Mereka membantu browser menemukan file HTML, gambar, CSS, JavaScript dan lainnya.

Ini adalah judul

Jenis jalur file:

  • Jalur absolut - URL lengkap (seperti https://www.example.com/images/picture.jpg)
  • Jalur relatif - Relatif terhadap halaman saat ini (seperti images/picture.jpg)

Contoh jalur file relatif:

./images/logo.png - file di folder images dalam folder saat ini
../images/logo.png - file di folder images satu level di atas folder saat ini

Elemen HTML <head> berisi metadata tentang dokumen HTML. Metadata tidak ditampilkan di halaman browser.

Ini adalah judul

Elemen yang biasanya ada di <head>:

  • <title> - Judul dokumen
  • <style> - Informasi style untuk dokumen
  • <meta> - Metadata tentang dokumen
  • <link> - Link ke sumber daya eksternal
  • <script> - Script/kode JavaScript
  • <base> - Menentukan URL dasar untuk semua URL relatif

<meta name="keywords" content="HTML, CSS, JavaScript">
<meta name="author" content="John Doe">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
</head>

Layout HTML mendefinisikan struktur dan tata letak halaman web. HTML menyediakan beberapa elemen semantik untuk mendefinisikan area pada halaman web.

Ini adalah judul

Elemen semantik dalam HTML5:

  • <header> - Header untuk dokumen atau bagian
  • <nav> - Container untuk link navigasi
  • <section> - Bagian dalam dokumen
  • <article> - Konten independen dan mandiri
  • <aside> - Konten sampingan
  • <footer> - Footer untuk dokumen atau bagian
  • <details> - Informasi tambahan yang dapat ditampilkan/disembunyikan
  • <summary> - Heading untuk elemen <details>
